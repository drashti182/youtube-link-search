# YouTube Link Searcher

A Python tool to fetch YouTube video links based on any search query.  
Search YouTube and get the top N video URLs quickly without opening a browser.

## Features
- Search YouTube by any keyword.
- Returns top video links (default 5, configurable).
- Easy to use as a script or in other Python programs.
